=head1 NAME

SNMP::Extension::PassPersist - Generic pass/pass_persist extension 
framework for Net-SNMP


=head1 DESCRIPTION

This module is a framework for writing Net-SNMP extensions using the
pass or pass_persist mechanisms.


=head1 INSTALLATION

To install this module, run the following commands:

    perl Makefile.PL
    make
    make test
    make install

Alternatively, to install with Module::Build, you can use the 
following commands:

    perl Build.PL
    ./Build
    ./Build test
    ./Build install
